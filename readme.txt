This Plugin gives the user the ability to embed the Last.FM mediaplayer on the sidebar. 

== Description ==

This plugin gives you the ability to use the Last.FM mediaplayer to play tracks from your favorite artist on your blog. Just enter the artist, check if you want to enable autostart and be happy :)

== Installation ==

1. Just upload the php file to your plugins directory
2. Now activate the plugin trough the "Plugins" menu and start using your new widget as usual
